# 星汉灿烂 1v1 模式拓展包

## 简介

星汉灿烂是新月杀的一个1v1竞技模式拓展包，基于官方比赛规则设计。

## 游戏规则

### 模式介绍

星汉灿烂1v1模式选手需要在每轮比赛中构建自己的武将池，通过对武将进行调配、组合，在多局对战中取胜。

### 玩法概括

- **五局三胜制**
- 每局比赛选手将构建自己的武将池，并在每小局中派出1-2名武将上场
- 胜方武将每小局对局获胜后将被锁定后续不可上场，败方武将可收回后续再用
- 当一方累计赢得了5名武将获胜时，该方获得本轮胜利

### 游戏流程

#### 1. 先手确认

选手入座，通过比点决定先后手，点数大的一方获得先手权，相同则重比。此后每小局先手、后手交替。先手方首轮摸牌阶段摸牌数减1。

#### 2. 禁将顺序

从公共武将牌堆中展示18张武将牌。根据先后手进行下方禁将步骤，每个步骤限制15秒，超时则默认当前选手最左侧的武将牌：

- 步骤1：后手方禁用武将1名，剩余可选武将数17名
- 步骤2：先手方禁用武将2名，剩余可选武将数15名
- 步骤3：后手方禁用武将1名，剩余可选武将数14名

#### 3. 选将步骤

当禁将完成后，根据先后手进行下方选将步骤，且选择武将需持续展示：

- 步骤1：后手方选择武将1名
- 步骤2：先手方选择武将2名
- 步骤3：后手方选择武将2名
- 步骤4：先手方选择武将2名
- 步骤5：后手方选择武将2名
- 步骤6：先手方选择武将2名
- 步骤7：后手方选择武将2名
- 步骤8：先手方获得剩余的1名武将

#### 4. 上阵武将选择

每小局选手从自己的武将池中自行决定派出1-2名武将上场（无需告知对手数量）。

#### 5. 分发起始手牌

每小局开始时双方选手各4张起始手牌，起始手牌的分发顺序为：先后手轮流从牌堆摸一张牌，直到双方手牌达到4张。

#### 6. 重整流程

每小局中当一方派出武将阵亡后若未满足胜利条件，则进入重整（即进入新一小局），双方派出新的武将上场。

### 胜利条件

当一方累计赢得5名武将获胜时，该方获得本局胜利。先赢得3局的选手获得最终胜利。

### 双将规则

选双将时，靠近体力牌的决定性别与势力，体力值与体力上限为二者之和的一半（向下取整），多余的勾玉没有补偿。

### 牌堆规则

每小局结束不洗牌，原牌堆用于下一小局比赛，当需要摸牌且牌堆剩余牌数不支持本次操作时才会进行洗牌。

### 鏖战规则

1. 当单局比赛牌堆进行第2次洗牌后，当前行动角色回合结束后，所有【桃】视为【酒】。
2. 当单局比赛牌堆进行第3次洗牌后或当轮瑞士轮时间达到50分钟后，当前角色回合结束后，每名角色回合结束后扣减1点体力（不会触发武将技能）。

## 安装方法

1. 将 `xinghan_canlan` 文件夹复制到新月杀的 `packages` 目录下
2. 重启游戏，在游戏模式中选择"星汉灿烂"

## 文件结构

```
xinghan_canlan/
├── init.lua                          # 拓展包入口
├── pkg/
│   └── xinghan_mode/
│       ├── xinghan_1v1.lua           # 游戏模式核心逻辑
│       └── rule_skills/
│           └── xinghan_1v1.lua       # 规则技能
├── qml/
│   ├── XinghanSelect.qml             # 禁将/选将界面
│   └── XinghanDeploy.qml             # 上阵选择界面
├── i18n/
│   └── en_US.lua                     # 英文翻译
├── image/                            # 图片资源
├── audio/                            # 音频资源
└── README.md                         # 说明文档
```

## 开发说明

### 依赖

- 新月杀游戏框架
- Lua 5.4
- Qt/QML (界面)

### 自定义修改

如需修改游戏规则，可编辑以下文件：

- `pkg/xinghan_mode/xinghan_1v1.lua` - 核心游戏逻辑
- `pkg/xinghan_mode/rule_skills/xinghan_1v1.lua` - 规则技能实现
